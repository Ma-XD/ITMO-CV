# Лабораторная 2 - DET

В практически любой точке современного города человека окружает множество номеров и надписей.

1. Обучите (или дообучите) нейронную сеть детектировать номера на изображениях (номер представляет собой последовательность длиной до 20 символов из цифр, прописных и строчных букв русского или английского алфавита, знаков дефис и слеш).
2. Сфотографируйте номера на улицах города (5-10 фотографий) (не из Интернета, проверим поиском по картинке) и запустите на них свою дообученную модель. Фотографии должны быть максимально разнообразными (крупные, средние, мелкие планы).
3. Оцените качество детекции с помощью метрик IoU, Precision, Recall, mAP на тестовой части набора The Street View House Numbers http://ufldl.stanford.edu/housenumbers/ (В формате для YOLO этот датасет есть на Roboflow). Требуется mAP не менее 0,6.
4. Детекция номеров можно улучшить, предобучив модель на датасете
https://universe.roboflow.com/university-of-toronto-xho85/numberdetection-eppfj/dataset/2
5. Сделать вывод
